# ReelBox 스프린트 할일

이 문서는 현재 프로젝트 상태를 기준으로, 주별로 작업을 쪼개서 진행하기 위한 실행 문서다.

기준 원칙
- 먼저 제품 설명과 실제 기능을 맞춘다.
- 다음으로 검색과 재탐색 경험을 강화한다.
- 마지막으로 운영 안정성과 데이터 품질을 높인다.

## 진행 방식 제안

매주 운영 방식
- 주 초에 해당 스프린트 범위를 확정
- 주 중에는 구현과 확인
- 주 말에는 실제 사용 관점으로 점검

주별 체크 질문
- 이번 주 변경이 제품 핵심 가치와 직접 연결되는가
- 실제로 다시 찾기 쉬워졌는가
- 데이터 품질이 좋아졌는가
- 다음 스프린트로 넘어가기 전에 막아야 할 위험이 남아 있는가
